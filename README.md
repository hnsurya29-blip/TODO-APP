# Todo List App

A simple Todo List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive user interface
- Dynamic DOM manipulation using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

Todo-App/
│
├── index.html
├── style.css
└── script.js

## How It Works

### Add Task
- Enter a task in the input field.
- Click the **Add** button.
- The task is added to the list.

### Complete Task
- Click on a task.
- The task is marked as completed with a strike-through effect.
- Click again to remove the completed status.

### Delete Task
- Click the **Delete** button next to a task.
- The task is removed from the list.
